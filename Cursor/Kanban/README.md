# Kanban Project Manager

A single-board Kanban web app built with Next.js.

## Run

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Test

```bash
cd frontend
npm test              # unit tests
npm run test:e2e      # Playwright E2E tests
```

## Features

- Five fixed columns (renamable)
- Cards with title and details
- Drag and drop between columns
- Add and delete cards
- Pre-populated dummy data (no persistence)
