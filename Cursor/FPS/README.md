# アリーナ FPS

矢印キーで移動し、スペースキーで発砲する3D一人称視点シューティングゲームです。

## 遊び方

1. `index.html` をブラウザで開く（または下記のローカルサーバーを使用）
2. 「バトル開始」をクリック
3. コンピュータ対戦相手を倒して勝利を目指す

## 操作方法

| キー | 動作 |
|------|------|
| ↑ | 前進 |
| ↓ | 後退 |
| ← | 左回転 |
| → | 右回転 |
| Space | 発砲 |

## ローカルサーバー（推奨）

ES Modules を使用しているため、ローカルサーバー経由での起動を推奨します。

```bash
npx serve .
```

ブラウザで `http://localhost:3000` を開いてください。

## 技術

- [Three.js](https://threejs.org/) — 3Dレンダリング
- バニラ JavaScript（ビルド不要）
